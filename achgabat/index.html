Achgabat development diary

Achgabat is a small 2D engine using SFML 2.4.2 and C++14, meant to be a base to 2D games and based of flexibility.

This page will act as a diary. Each major commit to the achgabat github will be followed by a post explaining the changes. You are welcome to provide commits and ideas!

The engine is experimental and require further SFML and C++ knowledge, there is no graphical interface or such. This isn't Unity or Cocos2D, if you don't know where to start, I suggest you take a look at what is SFML.

Tickers

Tickers are a simple way to make custom update cycles with custom tickrates in the main thread. The only parameter is the delta time. Their use is very simple : 

   
		ag::Ticker ticker; //declaration
		if(ticker.check(dt)) //dt is delta time (sf::Time)
        {
			//do stuff
			//this code will be executed every 1/40 of a second
        }
